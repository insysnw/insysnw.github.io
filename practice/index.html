Практика

Все приведённые на данной странице материалы направлены на получение практических навыков. К сожалению, лабораторные на данный момент требуют заранее подготовленной преподавателем инфраструктуры. Хорошая новость - создание инфраструктуры автоматизируется с помощью Pulumi. Это значит, по возможности мы выложим инструкции для подготовки инфраструктуры, ориентируясь на индивидуальное использование студентами и групповое другими преподавателями.

Домашние задания

Выполняются самостоятельно.

  • #1 TCP чат
  • #2 Реальный UDP протокол
  • #3 Разработка протокола
  • #4 OpenAPI спецификация

Лабораторные

Выполнятся всей группой в подготовленных преподавателем условиях.

  • HTTP
  • Эмуляция неблагоприятный условий сети
